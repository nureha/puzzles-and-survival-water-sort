# Water Sort Solver — パズルアンドサバイバル用

**[▶ アプリを開く](https://nureha.github.io/puzzles-and-survival-water-sort/)**

パズルアンドサバイバル（Puzzles & Survival）のウォーターソートパズルを解くWebアプリです。
盤面を入力するだけで最短手順を自動で算出します。

---

## 使い方

入力した盤面の情報はサーバーに送信されません。解の計算も含めすべての処理がお使いの端末のブラウザ内で完結します。

### 1. 本数を設定する

画面上部の「＋」「−」ボタンで試験管の本数を調整します。

### 2. 盤面を入力する

凡例（図形とアルファベットの対応表）を参考に、ゲーム画面の各試験管の内容を上から順にアルファベットで入力します。

- 色が判明していないセルは `?` を入力します
- `?` を入力したセル以降は自動的に `?` で埋まります（それより下の色は見えないため）

### 3. 「解く」ボタンを押す

手順が見つかった場合、移動手順の一覧が表示されます。

### 4. 手順をタップして盤面を進める

手順の各ステップをタップ（クリック）すると、盤面がその手順の状態に進みます。もう一度タップすると1手戻ります。実際のゲームと見比べながら操作を進めてください。

---

## 色が不明な場合（? の使い方）

試験管の上に液体が重なっていて下の色が見えないときは `?` を入力します。

- **推定解**: `?` が含まれていても、ランダムに色を仮定して解を試みます。うまく解けた場合は手順が表示されますが、実際の色と異なる場合は手順の途中で行き詰まることがあります。
- **部分解**: 推定が難しい場合は、`?` の色を明らかにするための手順がヒントとして表示されます。表示された手順を実行することで隠れた色が判明し、改めて入力して解くことができます。

---

## 保存 / 読み込み

「保存 / 読み込み」ボタンから、現在の盤面に名前をつけて保存できます。保存したデータは後から読み込んで再利用できます。

**クリア済みステージの登録にも活用できます。** 手順を最後まで進めてクリア状態になったら、その状態を保存しておくことで、後から振り返ることができます。

データはサーバーには送信されず、お使いの端末のブラウザ内にのみ保存されます。

---

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動（http://localhost:5432）
npm run build    # プロダクションビルド（dist/）
```
